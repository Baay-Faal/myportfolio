## 🎨 Charte Graphique & Design

Le site est conçu selon les principes de design haut de gamme de Nike :
*   **Palette de couleurs** : Bleu Azur (couleur d'accent dynamique), nuances de gris (fond et cartes en gris clair, textes structurés en gris foncé) et blanc pur.
*   **Aesthétique minimaliste** : Typographie lourde et affirmée, larges espaces blancs, lignes géométriques nettes et ombres portées discrètes pour un effet de relief 3D.
*   **Zéro émoji** : Une interface 100% sobre et axée sur le professionnalisme.
*   **Logos & Icônes** : Utilisation des logos de marques officiels et de tracés SVG vectoriels de haute précision.

---

## 💻 Technologies Clés du Portfolio

Le portfolio est développé en technologies web natives pour garantir des performances optimales et un chargement ultra-rapide :
*   **Structure** : HTML5 sémantique optimisé pour le référencement (SEO).
*   **Stylisation** : CSS3 Vanilla unifié dans un fichier unique `style.css` pour une cohérence globale du système de design (variables CSS, flexbox, grid 3D).
*   **Interactivité** : JavaScript Vanilla (`script.js`) pour la navigation mobile et les animations de défilement (Scroll Animations).

---

## 📂 Structure Multi-Pages

Le site est découpé en plusieurs pages dédiées :
1.  **Home (`index.html`)** : Section d'accueil comprenant un bandeau de statistiques clés, une présentation de ma philosophie de conception (Performance, Architecture Clean, Minimalisme) et la mise en valeur de mon projet phare (**EventPass**) avec un visuel de ticket d'accès VIP interactif en SVG.
2.  **À Propos (`about.html`)** : Présentation de mon profil de développeur, de mon cursus académique, de ma passion pour les jeux vidéo et de l'origine de mon alias de joueur `</Baay_Hacker>` (alliant les valeurs de travail et d'humilité du mouvement *Baay Faal* au Sénégal à ma passion pour la technique informatique).
3.  **Projets (`projects.html`)** : Galerie de mes réalisations techniques avec filtres interactifs et badges de catégorie.
4.  **Parcours (`skills.html`)** : Frise chronologique 3D de mes certifications et compétences classées (Langages, Frameworks, Bases de données, Outils).
5.  **Contact (`contact.html`)** : Formulaire de contact moderne et liens directs vers mes canaux professionnels (Email, LinkedIn, WhatsApp).

## 🛠️ Installation & Lancement Local

Pour visualiser le portfolio localement :

1.  Clonez ce dépôt Git :
    ```bash
    git clone https://github.com/Baay-Faal/myportfolio.git
    ```
2.  Accédez au dossier du projet :
    ```bash
    cd myportfolio
    ```
3.  Lancez un serveur HTTP local (par exemple avec Python) :
    ```bash
    python -m http.server 8000
    ```
4.  Ouvrez votre navigateur à l'adresse suivante :
    `http://localhost:8000/index.html`
